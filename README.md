# Text-to-Image AI

**Text-to-Image AI** is a simple tool that leverages ChatGPT and DALL-E to generate images from text. Users can either select an art style from the provided options or input a custom style, then hit "Generate" to create their image.

---

## Features

- Input text prompts to generate images.
- Choose from predefined art styles or input your own.
- React-based frontend with a clean and simple interface.

---

## Demo

You can check out the live project here: [Text-to-Image AI](https://simple-text-to-image-bot.web.app/)

---

## Tech Stack

- **Frontend Framework**: React
- **API Communication**: Axios
- **Styling**: Material UI (MUI)

---

## Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Kidker95/text-to-image-ai.git
   ```
2. Navigate to the project directory:
   ```bash
   cd text-to-image-ai
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to use the app.

---

## Folder Structure

- **src**: Contains all source files.
  - **components**: React components.
  - **services**: Axios services for API calls.
  - **utils**: Utility functions and configurations.
- **build**: Production-ready files generated after build.
